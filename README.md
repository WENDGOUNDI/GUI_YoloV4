# GUI_YoloV4
The aim of this project is to build a GUI for object detection app using YoloV4 pretrained weights on COCO dataset. Later the GUI has been converted into a python .exe app.

# Dependencies
 - IO
 - OS
 - PySimpleGUI
 -PIL
 -CV2
 - Pyisntaller
 - Auto-py-to-exe
